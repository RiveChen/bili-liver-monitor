# bili-liver-monitor

Bilibili 主播直播状态监控与推送工具。

All codes are vibed.

## 功能

- [x] 开播/下播提醒（HTTP 轮询）
- [x] Bilibili 动态推送
- [x] 微博动态推送
- [x] NapCat Bot 命令
- [ ] WebSocket 实时监听（开播检测/礼物/SC）
- [ ] 直播封面推图
- [ ] Bark iOS 告警通道
